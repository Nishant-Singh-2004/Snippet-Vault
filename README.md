# 🚀 SnippetVault

A modern full-stack code snippet manager where users can create, share, and explore code snippets with syntax highlighting.

## 🔗 Live Demo

👉 [Live Link](https://snippet-vault-peach.vercel.app/)

---

## ✨ Features

- 🔐 Authentication (Signup/Login)
- 🧠 Create, edit, delete snippets
- 🌍 Public snippet sharing
- 🔎 Search by title, language, and tags
- 🏷️ Tag-based filtering
- 👤 User profiles (`/u/[username]`)
- 🔗 Share snippet links
- 📋 Copy code button
- 🎨 Syntax highlighting
- 🖼️ Export snippet as image
- ⚡ Live updates using React Query

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router)
- **Backend:** Supabase (Auth + Database)
- **State Management:** TanStack Query
- **Styling:** CSS Modules
- **Deployment:** Vercel
